```

```

## Использованный стек
- Платформа: .NET 6.0 (LTS)
- Фреймворк: ASP.NET Core MVC
- ORM: Entity Framework Core 6.0.36
- База данных: EF Core InMemory Database / SQL Server 
- Фоновые службы: BackgroundService (IHostedService)
- Язык: C# 10.0
- Архитектура: MVC + REST API
